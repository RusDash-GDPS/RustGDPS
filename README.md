RusDash GDPS - это GDPS проект, целью которого является показать русские фишки, принципы и мнения путём игры на сервере.

На RusDash работает Globed сервер, с помощью которого вы можете играть вместе с любого устроиства, из любой точки мира. Так же на RusDash присутствуют свои уникальные уровни, маппаки, гаунтленты, свободные критерии оценивания, полная безопасность для игроков и это не всё! Специально для удобства своих игроков команда RusDash сделала множество полезных разработок, среди которых: форум для игроков прямо на официальном сайте; чат для игроков на официальном сайте (для тех, у кого, возможно нет социальных сетей); прошитый пакет модификаций, с помощью которого вы можете настроить RusDash на свой вкус и предпочтения.

Для начала игры на сервере RusDash вы можете воспользоваться модом GDPS Switcher или скачать RusDash с официального сайта (находится в профиле, так же является ссылкой для настройки конфигурации на GDPS Switcher). Пару шагов и вы - счастливый игрок на проекте RusDash, если вы не уверены в начале игры в RusDash, посмотрите раздел "особенности" на официальной странице проекта.
